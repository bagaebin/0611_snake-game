# Development Tasks

1. **프로젝트 스캐폴딩**
   - index.html, style.css, script.js 기본 구조 작성
   - Three.js CDN 및 폰트, 메타 태그 삽입

2. **씬 초기화**
   - 렌더러, 씬, 카메라, 조명, 캔버스 크기 조정 로직 구현
   - 16x16 격자 평면과 보조 격자 라인 생성

3. **HUD 및 UI 구성**
   - 점수 표시, 센서 안내, 권한 요청 버튼 레이아웃 배치
   - 반응형 스타일 적용

4. **입력 컨트롤러 작성**
   - DeviceOrientation 이벤트 처리 및 권한 요청 로직 구현
   - 카메라 기준 우/하 벡터를 계산해 기울기 방향을 지면에 투영하고 데드존/보간 처리
   - 좌우 화살표 키(왼쪽=시계, 오른쪽=반시계) 입력으로 진행 방향을 상대적으로 회전시키는 각속도 로직 구현

5. **스네이크 초기화**
   - 무작위 시작 위치 계산 (격자 범위 내)
   - 헤드 메시 생성 및 경로 버퍼 초기화
   - 세그먼트 리스트 및 길이 관리 구조 작성

6. **이동 및 경로 버퍼 로직**
   - 매 프레임 헤드 이동 계산 (deltaTime 기반)
   - 경로 누적, 거리 계산, 오래된 경로 정리
   - 세그먼트 위치 보간 함수 구현
   - 경계면과 충돌 시 유효한 진행 벡터 성분만 유지해 벽을 따라 미끄러지는 보정 처리

7. **코인 시스템 구현**
   - 타이머 기반 코인 스폰 로직 및 빈 셀 찾기
   - 수집 감지 및 점수/세그먼트 업데이트

8. **게임 루프 통합**
   - `requestAnimationFrame` 루프에서 이동, 코인, 렌더링 호출
   - HUD 업데이트 및 경계 체크
   - 자기 충돌 감지(단, 두 번째 세그먼트와의 접촉은 허용) 및 게임오버 오버레이/재시작 흐름 구현

9. **테스트 및 튜닝**
   - 모바일 기기에서 기울기 민감도 조정
   - 데스크톱 키보드 조작 확인
   - 성능 검토 및 필요 시 최적화

10. **문서 및 배포 준비**
    - README/가이드 문구 검토
    - 배포용 경량화 설정 확인
